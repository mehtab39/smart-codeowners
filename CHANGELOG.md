# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2024-01-15

### Added
- Initial release of smart-codeowners
- Git commit history analysis
- Recency weighting with configurable half-life
- Clear majority vs top contributors detection
- Folder-level ownership consolidation
- Smart filtering (merge commits, bot commits, bulk changes)
- CLI commands: `generate`, `analyze`, `init`
- Configurable thresholds and options
- GitHub CODEOWNERS format output
- Email to GitHub username mapping
- Comprehensive documentation
- Test suite for core functionality
